# MyKvadro вҖ” бғҷбғ•бғҗбғ“бғ бғқбғӘбғҳбғҷбғҡбғ”бғ‘бғҳбғЎ бғӣбғҗбғ бғҷбғ”бғўбғҳ (ATV Marketplace)

A full-stack ATV/UTV marketplace built with **Next.js 16 (App Router)**, **TypeScript**,
**Tailwind**, and a **MySQL** backend served from internal Next.js API routes. It supports
public browsing, listing detail pages with SEO + sharing, user accounts, a sell/edit flow,
and a full admin panel for managing products, brands, and categories.

## Architecture

- **Frontend** вҖ” React 19 client pages + server-rendered ATV detail pages.
- **Backend** вҖ” Next.js Route Handlers under `app/api/**` (no separate server needed).
- **Database** вҖ” MySQL/MariaDB. Schema lives in the `atv` database (Laravel-origin schema:
  `users, brands, categories, locations, atvs, atv_images, blogs`).
- **Data access** вҖ” `lib/db.ts` (a pooled `mysql2` client) + `lib/api.ts` helpers.
- **Auth** вҖ” stateless HMAC-signed bearer tokens (`lib/auth.ts`), bcrypt passwords
  (compatible with existing Laravel `$2y$` hashes).
- **Images** вҖ” fully self-contained: themed SVGs generated by the seed into `public/seed/`
  (one gallery per ATV, a logo per brand, a tile per category). Admin uploads land in
  `public/uploads/`.

## Setup

1. **MySQL** must be running with an `atv` database (configure via `.env`):

   ```
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_USER=root
   DB_PASSWORD=
   DB_NAME=atv
   AUTH_SECRET=<change-me>
   NEXT_PUBLIC_SITE_URL=http://localhost:3000
   ```

2. **Seed** the database with mock data + generated images:

   ```bash
   yarn seed          # 60 ATVs В· 16 brands В· 8 categories В· blogs В· users
   ```

3. **Run** the dev server:

   ```bash
   yarn dev           # http://localhost:3000
   ```

## SEO & performance

- Per-listing dynamic metadata + OpenGraph/Twitter cards + JSON-LD `Product` schema.
- `app/sitemap.ts` (includes every active ATV) and `app/robots.ts`.
- Static prerendering for content pages; on-demand rendering for data pages & API.
- Tasteful entrance/hover animations (Tailwind keyframes) with `prefers-reduced-motion`
  support.
